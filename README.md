I HAVE KU (WEB ECOMMERCE)

โปรเจคนี้เป็นเว็บแอปพลิเคชัน E-Commerce สำหรับขายสินค้าในมหาวิทยาลัยเกษตรศาสตร์ พัฒนาด้วยสถาปัตยกรรม Full-Stack โดยแบ่งออกเป็น 2 ส่วนหลัก คือ 
Frontend ที่พัฒนาด้วย Vue.js และ 
Backend ที่พัฒนาด้วย Node.js/Express เชื่อมต่อกับฐานข้อมูล PostgreSQL ระบบรองรับการทำงานของทั้งผู้ใช้ทั่วไปและผู้ดูแลระบบ (Admin) 
โดยมีฟีเจอร์หลักได้แก่ การแสดงสินค้า, การค้นหาและกรองสินค้าตามหมวดหมู่, ระบบตะกร้าสินค้า, ระบบสมาชิก (Register/Login), และ Admin Panel สำหรับจัดการสินค้าและสมาชิก

Tech Stack:

Frontend: Vue.js, Pinia (State Management), Bootstrap
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT (Secret Key)
Tools: Nodemon, Dotenv, Swagger (API Docs)
